# Requirements
- Git / GitHub CLI
- GCC
- Make

To run it locally:
```
// using git with HTTPS
git clone https://github.com/R3ZV/Sorting-Visualizer.git

// using git with SSH
git clone git@github.com:R3ZV/Sorting-Visualizer.git

// usin GitHub CLI
gh repo clone R3ZV/Sorting-Visualizer

cd Sorting-Visualizer

make
```
